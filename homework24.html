Visual Studio Code

Visual Studio Code or Code is a source code editor developed by Microsoft for Microsoft Windows, macOS, and Linux. Debugging support, Git control, syntax highlighting, etc. are included, and the user can modify the theme, shortcut keys, and settings of the editor.

Languages: English, French, German, Italian, Japanese, Korean, Russian, Spanish, Traditional Chinese
